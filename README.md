# 🚀 Full-Stack Developer Portfolio

A modern portfolio website showcasing real-world systems, enterprise applications, and practical software solutions.

## 🌐 Live Demo

ssenkungu.vercel.app

---

## 🧠 About

This portfolio highlights my experience building scalable systems across finance, NGOs, and enterprise operations. It focuses on practical, real-world applications rather than just demos.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Backend/API:** Next.js API routes
* **Email Handling:** Resend
* **Content:** MDX (for blog & projects)
* **Deployment:** Vercel

---

## 📂 Features

* 📌 Project showcase with detailed case studies
* ✍️ Blog section (MDX-powered)
* 📬 Functional contact form (email integration)
* 🌙 Dark/light theme toggle
* ⚡ Fast, responsive UI

## 🚀 Getting Started

Clone the repo:

```
git clone https://github.com/dssenkungu/Portfolio-Website.git
cd Portfolio-Website
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env.local` file and add:

```
RESEND_API_KEY=your_api_key_here
```
